# tuomasbot

Telegram chatbot, jonka avulla pystyy seuraamaan rahan käyttöä. Botille syötetään maksun tiedot, jotka se tallentaa tiedostoon. Menneitä maksuja voi kysyä botilta eri parametreillä. Botti on tarkoitettu useamman henkilön käytettäväksi jolloin voidaan seurata maksuja esimerkiksi kaveriporukan tai pariskunnan kesken.
